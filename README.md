# 学习OS

一个为 ADHD 考生设计的**外部执行功能系统**。不是番茄钟，不是待办清单，不是时间管理工具。

**核心思路：管理启动阻力而非管理时间，管理情绪而非管理任务，管理连续性而非管理效率。**

## 快速开始

下载 `study-os-standalone.html`，双击打开。不需要安装，不需要网络，数据完全存储在浏览器本地。

## 怎么用

每天早上坐到书桌前，花 10 秒选一个身体信号——**平静 / 焦虑 / 无聊 / 疲惫 / 兴奋**。系统根据信号自动决定今天几张卡（1~5 张）。每张卡只写一行——一个你能在 10 秒内开始的物理动作。

点击卡片，开始 50 分钟冲刺块。块内不离开、不换科、不碰手机。铃响进入 10 分钟硬休息。

块数是唯一 KPI。不统计时长，不统计页数。

## 主要功能

- **信号系统** — 先读身体，后选任务。五种心情对应五种任务量
- **启动卡** — 禁止模糊动词，只写物理动作。10 秒启动测试
- **冲刺块** — 50+10 不可中断。三种结局：未完/完成/卡太小
- **任务池** — 弹药库，最多 20 张。14 天不动的卡自动提醒
- **接触区** — 0~6 级渐进暴露，专攻高抵触科目。允许从最低开始，允许在最低停止
- **后备卡** — 4 张无脑续命卡。提前完成不需要思考"现在该做什么"
- **链图** — 唯一不可丢失的东西。非零日宪法：任何一天不允许归零
- **崩溃恢复** — 4 个预写密封信封。崩溃时不要自己想办法，让冷静时的你做决策
- **自动日夜模式** — 日落后自动切换深色主题
- **数据导出/导入** — JSON 备份，你的数据只属于你

## 技术说明

单文件 HTML 应用。纯 vanilla JS，无框架，无构建工具，无外部依赖。数据存储在 `localStorage`，不上传任何服务器。PWA 支持，可添加到手机主屏幕离线使用。

## 适用人群

ADHD 考生、长期备考者、对执行功能有需求的人。如果你经常坐在桌前不知道从哪开始，或者学了一小时不知道学了什么——这个系统是为你设计的。
